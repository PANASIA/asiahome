Piloting

In Piloting an assessment and a description of the organizational impact caused by the designed Business Control Model are made. This stage results in the final business model.

pO4 - Organizational Changes Approved

pM4 - Organization Model Defined

pP4 - Key Users Trained (details)

pB3 - Data Coding Schemes Defined

pC3 - Customizations/Localizations Completed

pI3 - Interfaces Completed

pM5 - Documentation Completed

pO5 - Processes and Organizational Documentation Generated

pT3 - Data Conversion Plan Defined

pT4 - Information System Prepared

pC4 - Low Priority Customizations/Localizations Specified

pB4 - Final Business Solution Agreed (simulation 2)
