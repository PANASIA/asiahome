Milestone Library

All Milestones

In this view all milestones are given according to the most comprehensive program of a Baan implementation.

By Scenario

A scenario is a predefined set of milestones that support a project manager in initiating and planning a project initiated by the Target Enterprise program.

By Module

Modules are the stage oriented milestone libraries of Target Enterprise. Target Enterprise consists of a Base module supplemented by several add-ons.

Implementation Lifecycle

The implementation lifecycle gives a high level structure of a Target Enterprise program in which a specific phase and stage must be selected.
