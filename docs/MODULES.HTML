Modules

Modules are the stage oriented milestone libraries of STACIS(R) Chamber. STACIS(R) Chamber consists of a Base module supplemented by several add-ons.

  • Base Module

The STACIS(R) Chamber Base module is a set of milestone definitions for the selection and implementation of UDS IEA solutions, based on an enterprise model with a good fit.

  • Customizations & Interfaces

The Customizations and Interfaces module is a set of milestone definitions for the customization and implementation of the necessary interfaces to the UDS software. Customizations and Interfaces is an add-on module whose input and output is directly related to the Base module.

  • Model Enhancements

The Model Enhancements module is a set of milestone definitions for the development and enhancement of Business models based on the Enterprise Architect Modeling concept. Model Enhancements is an add-on module. The inputs and outputs are directly related to the Base module.

  • Competence Center

The Competence Center module is a set of milestone definitions for the set up of a competence center. A competence center is a formal or virtual organization entity that keeps competencies available within the customer organization. The competence center module is an add-on, which is not directly linked to the Base module.
