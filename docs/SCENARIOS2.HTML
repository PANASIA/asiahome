Scenarios

A scenario is a predefined set of milestones that support a project manager in initiating and planning a project initiated by the Target Enterprise program.

  • s1 - Select a Baan Solution

Select Baan Solution is a predefined set of milestones that supports a customer in the introduction of Baan, the prospect and the mutual qualification of the potential business solution, cultural compatibility, and technical viability of a Baan implementation in the enterprise.

  • i1 - Prepare a Corporate Kernel Model

Prepare Corporate Kernel Model is a set of predefined milestones for building a business model of the customer organization on corporate and global level.

  • i2 - Implement Based on Standard Model

Implementation based on Standard Model concerns a milestone plan for an implementation based on a reference model or kernel model.

  • i3 - Implement Based on Customized Model

Scenario I3 is meant for implementation based on a model that is derived from a reference model or corporate kernel with less than 100% fit.

  • i4 - Prepare Model and Implement

Prepare Model and Implement describes a single site implementation including the building, piloting and implementation of a business model in a situation where a reference model or kernel model is not used.

  • c1 - Define and Implement Competence Center

Define and implementation Competence Center is a milestone approach for setting up a project for the implementation of a Competence Center.
