Phasing

The implementation lifecycle gives a high level structure of a STACIS(R) Chamber program in which a specific phase and stage must be selected.

  • Selection

Selection includes a set of milestones required to configure the business solution and the functional solution for the customer.

  • Visioning

Visioning initiates design and builds the company’s vision into a business model.

  • Proof of Concept

Proof of Concept refers to the pilot implementation to test the results of the previous phase in a real life situation.

  • Implementation

Implementation includes the common mapping, piloting and migration stages performed with a minimum or a maximum set of milestones, and concerns the actual roll out for the remaining sites.

  • Continuous Improvement

In future versions of STACIS(R) Chamber Continuous Improvement will be available.
