Migration

Migration is the stage in which the systems and data are transferred and the system, personnel and organization become operational.

iT5 - Information System Configured

iB5 - Master Data Entered and/or Converted

iC5 - Low Priority Customizations/Localizations Completed

iP5 - End User Training Completed

iB6 - Information System Accepted (simulation 3)

iO6 - Organizational Changes Implemented

iB7 - Information System Operational

iO7 - Project Closed
