Introduction

The Introduction gives a high level overview of Target Enterprise. It also describes the new features and gives a short outline of the structure of Target Enterprise.

Concepts

In this document the key concepts of Target Enterprise are described. Target Enterprise is structured around milestones and contains tools, documentation, and a method for program management.

White Paper & Overview

The White Paper gives a brief description of all elements included in Target Enterprise. It also provides information about who to contact for questions or future releases.

Program Management

The Program Management is an add-on module for multi-site implementations. Because it provides an essential framework for the different projects that have to be carried out, it is recommended to read this document first in case of a large or multi-site implementation.

Base Module

The Target Enterprise Base module is a set of milestone definitions for the selection and implementation of Baan ERP solutions, based on an enterprise model with a good fit.

Customizations & Interfaces

The Customizations and Interfaces module is a set of milestone definitions for the customization and implementation of the necessary interfaces to the Baan software. Customizations and Interfaces is an add-on module whose input and output is directly related to the Base module.

Model Enhancements

The Model Enhancements module is a set of milestone definitions for the development and enhancement of business models based on the Dynamic Enterprise Modeling (DEM™) concept. Model Enhancements is an add-on module, whose input and output is directly related to the Base module.

Competence Center

The Competence Center module is a set of milestone definitions for the set up of a Competence Center. A Competence Center is a formal or virtual organization entity that keeps competencies available within the client organization. The Competence Center module is an add-on that is not directly linked to the Base module.

Project Configurator

The Project Configurator is a knowledge-based tool used to configure all milestone and activity information for a customer project. It provides reporting capabilities such as the Plan of Approach, project schedules, Milestone and Responsibility charts and Budget Proposals.

Microsoft Team ManagerTM

Microsoft Team ManagerTM is a tool that provides a way for the Project Manager and his team members to communicate about the ongoing work. Individual task lists are used to keep track of planed activities and report in a uniform way about the work in progress. It fills in the gap between Microsoft Project and Baan hours accounting.
